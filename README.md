# 雑学ストックアプリ「明日、だれかに話したい」
「自分の話」をするのが苦手なすべての人へ。
デモサイトはこちら: https://tomonea.pythonanywhere.com

雑談や自己開示が苦手な人のための、読書などで得た知識をストック・活用するWebアプリケーションです。

## 概要
日常会話で「最近どう？」と聞かれた際、自分のことを話すのが苦手なために会話に詰まってしまう悩みを解決するために開発しました。
自分の体験談ではなく、「本で読んだ面白い話（雑学）」をストックしておき、会話のネタとして提供することで、楽しくスムーズなコミュニケーションを支援します。

## 機能
- **雑学ストック機能**: タイトル、内容、カテゴリ、出典を登録できます。
- **活用記録機能**: 実際に会話で使用した回数をカウントできます（「話せた！」ボタン）。
- **一覧表示・削除**: ストックした知識をカード形式で管理できます。

## 使用技術
- Python 3.13
- Flask (Webフレームワーク)
- SQLite (データベース)
- HTML / CSS / JavaScript (標準ライブラリのみで実装)

## 起動方法（ローカル環境）

1. リポジトリをクローンまたはダウンロードします。
2. 必要なライブラリをインストールします。
    pip install -r requirements.txt
3. データベースを初期化します（初回のみ）。
    python3 init_db.py
4. アプリケーションを起動します。
    python3 init_db.py
5. ブラウザで http://127.0.0.1:8000 にアクセスしてください。

## こだわりポイント
- **UI/UX**: 手元ですばやく追加できるよう、入力フォームを画面遷移なしの下部ドロワー形式にしました。
- **モチベーション設計**: 「話せた回数」を可視化することで、インプット（読書）とアウトプット（会話）のサイクルが楽しくなるようにしました。
